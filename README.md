## ❌ macOS 支持说明

由于苹果系统的技术限制，本项目暂不支持macOS：

### 技术限制
- 🚫 无法使用KVM硬件加速，虚拟化性能极差
- 🚫 系统沙盒限制硬件访问和进程管理  
- 🚫 公证和签名要求增加分发复杂度

### 替代方案
- ✅ 在mac上安装Linux双系统
- ✅ 使用UTM等专为mac优化的虚拟化方案
- ✅ 通过云服务器访问完整的虚拟化功能
